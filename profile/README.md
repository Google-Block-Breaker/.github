## Installation

Follow these steps to run the game locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/google-block-breaker-game.git
   ```

2. **Navigate to the project folder**:
   ```bash
   cd google-block-breaker-game
   ```

3. **Open the game in your browser**:
   Simply open the `index.html` file in your preferred web browser.

---

## How to Play

1. Use your mouse or arrow keys to move the paddle left and right.
2. Bounce the ball to break the blocks.
3. Clear all blocks to advance to the next level.
4. Avoid letting the ball fall below the paddle!
